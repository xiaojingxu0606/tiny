拦截器类Inteceptor,添加拦截器时继承该类

Example
class Pre extends Ti.$http.Interceptor {
   constructor() {
     super();
   }
   resolve(request) {
     request.setData('me', '1');
     return request;
   }
}
Ti.$http.preInterceptor.add(new Pre());
