Ti. Component

组件类Ti.Component,所有组件必须继承该类 且必须通过super()调用父级构造器方法

Constructor

new Component(data)

Source:
  • component.js, line 29
Example
// 用法如下:
import html from '/path/template';
import style from '/path/style';

class ExampleComponent extends Ti.Component {
  constructor(data) {
    super(data);
  }
  render() {
    // 这里返回你的模板,可通过import从外部引入
    return html;
  }
  style() {
    return style;
  }
}

mounted()

Source:
  • component.js, line 86

组件方法mounted 该方法在组件加载完成后调用, 常用于初始化加载服务器的数据

Example
class App extends Ti.Component {
 mounted() {
   Ti.$http.get('/get', {responseType: 'json'})
   .then(function(response) {
     this.list = response.data;
   })
   .catch(function(err){
     console.log(err);
   });
 }
}
